# Welcome!

Tycho's Treasury is a collection of [Vue](https://vuejs.org/) components that implement an easily customisable design system, initially developed for use on my personal and professional websites.

It is comprised of:
- Design tokens: CSS variables that ensure consistency across the UI and can easily be customised on a per-project basis.
- Atoms: Foundational generic Vue components that are not exported or intended for direct use in websites/applications, but are shown here for reference and testing.
- Components: Reusable Vue components that are exported for use in websites/applications.

The library is named after my dog, Tycho - a natural continuation since my PHP UI library, [Comet Components](https://cometcomponents.io), is named after my older dog.
